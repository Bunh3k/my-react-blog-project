# 📝 RealWorld Blog Platform

A full-featured blog platform built with React, implementing authentication, article management, favorites, and user profiles.

---

## 🚀 Live Demo

👉 https://react-blog-project-test.vercel.app

---

## 🛠 Tech Stack

- React (Vite)
- React Router
- React Hook Form
- ESLint
- REST API (RealWorld API)

---

## ✨ Features

- 🔐 Authentication (Sign up / Sign in / Logout)
- 👤 User profile & settings
- 📝 Create / Edit / Delete articles
- ❤️ Favorite / Unfavorite articles
- 👥 Follow / Unfollow users
- 📰 Personalized feed (articles from followed users)
- 🏷 Dynamic tags from API
- 📄 Pagination
- ⚡ Loading states & error handling
